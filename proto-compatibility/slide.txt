# Protobuf Compatibility

## Start with Basic

v1
```proto
message HelloRequest {
    string name = 1;
}

message HelloReply {
    string message = 1;
}

service HelloService {
    rpc Hello(HelloRequest) returns (HelloReply);
}
```

## Add a Field

v2
```proto
message HelloReply {
    string message = 1;
    string version = 2;
}
```

## Rename a Field

v3
```proto
message HelloRequest {
    string alt_name = 1;
}
```

## Change a Field Type

v4
```proto
message HelloRequest {
    int64 name = 1;
}

```

## Delete a Field

v5
```proto
message HelloRequest {
    // string name = 1;
}
```

## Rename a Message

v6
```proto
message HelloV6Request {
    string name = 1;
}

service HelloService {
    rpc Hello(HelloV6Request) returns (HelloReply);
}
```

## Summerize

|Action               | Compatible |
|---                  | ---        |
| Add a Field         | Yes        |
| Rename a Field      | Yes        |
| Change a Field Type | No         |
| Delete a Field      | Yes        |
| Rename a Message    | Yes        |
